# Sea Level Predictor

## 📌 Description

A Python project that analyzes historical sea level data and predicts future sea level rise using linear regression. The project visualizes past trends and forecasts sea levels up to the year 2050.

This project was completed as part of the **freeCodeCamp Data Analysis with Python Certification**.

## 🎯 Objective

The objective of this project is to practice:

* Data Analysis with Python
* Working with real-world environmental datasets
* Data Visualization using Matplotlib
* Predictive Modeling with Linear Regression
* Exploratory Data Analysis (EDA)

## 🛠️ Technologies Used

* Python

## 🚀 Features

* Loads and analyzes historical sea level data from the EPA dataset.
* Creates a scatter plot of sea level measurements over time.
* Builds a line of best fit using the entire dataset.
* Builds a second line of best fit using data from the year 2000 onward.
* Predicts sea level rise through the year 2050.
